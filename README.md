# plotly-nlp-news

## Instruction

```bash
python3 -m venv .venv
source .venv/bin/activate
```

Install the requirements:

```
pip install -r requirements.txt
```
Run the app:

```
python app.py
```
You can run the app on your browser at http://127.0.0.1:8050
